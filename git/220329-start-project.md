# 2022-03-29 git learning

## Shell Command

```
cd Documents/			// Documents 폴더로 이동
mkdir dev			// dev폴더 생성
cd ..				// 하위 폴더로 이동
pwd				// 현재 작업위치 확인

touch readme.md			// readme.me 파일 생성
mv readme.md bin/		// bin폴더에 readme.md 파일 이동
cp readme.md bin/		// bin폴더에 readme.md 파일 복사
mv readme.md ./README.txt 	// 현재 폴더에 readme.md파일을 README.txt파일로 변환  
rm README.txt			// README.txt 파일 삭제

rm -rf bin/			// bin폴더 바로삭제
chmod 750 readme.md		// readme.md 파일의 권한을 750으로 변경
cat readme.md			// readme.md의 내용을 출력
vi readme.md			// Vim 에디터로 readme.md파일 열기

```
